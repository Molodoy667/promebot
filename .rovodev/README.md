# Rovo Dev Instructions

## Використання

### Автозапуск промту
Щоб застосувати стиль роботи з `PROMT 1` на початку сесії:

```powershell
# Windows PowerShell
.\.rovodev\autorun.ps1

# Або вручну
rovo run promt-1
```

### Доступні промти
- **PROMT 1** - Стиль роботи з мінімальними токенами, короткі відповіді, фокус на код
