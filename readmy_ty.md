# Описание проекта

## Назначение сайта
Одностраничный сайт, который выводит крупную надпись:

> **«Привет, мир»**

Сайт служит минимальным примером проекта на основе Lovable (React + Vite + Tailwind CSS).

## Технологии
- **React 18** — интерфейс
- **Vite** — сборка проекта
- **Tailwind CSS** — стилизация
- **Lovable** — онлайн‑редактор и хостинг проекта

## Структура главной страницы
- Файл: `src/pages/Index.tsx`
- Содержимое:
  - Корневой элемент `<main>` на всю высоту экрана
  - По центру расположен заголовок `<h1>` с текстом «Привет, мир»

## SEO и мета‑данные
Мета‑информация указана в `index.html`:
- `<title>Привет, мир</title>`
- `<meta name="description" content="Одностраничный сайт с надписью Привет, мир" />`
- Open Graph теги для корректного превью при шаринге в соцсетях.

## Интеграция с GitHub
Проект подключён к GitHub через интерфейс Lovable:
- Любые изменения, внесённые в Lovable, автоматически пушатся в подключённый репозиторий.
- Изменения, которые вы пушите из локального окружения в GitHub, автоматически подтягиваются обратно в Lovable.

## Как расширять сайт
Возможные направления развития:
- Добавить стилизованный фон и анимацию текста «Привет, мир»
- Создать дополнительные секции (о проекте, контакты и т.п.)
- Добавить навигацию и несколько страниц
- Подключить backend через Lovable Cloud для работы с данными и авторизацией
